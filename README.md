# Game Asset Formats

This repository contains a collection of game-oriented file formats 
and the tools used to create, inspect, and convert them, created by
me, with the specific intent of being used in games. This repository
is more or less for me (future reference, documentation, etc.), but
feel completely free to use them as you wish!

## Licensing

The documentation and tools are licensed under the MIT license. The
file formats are intended to be freely implemented by anyone.

## Formats

So far, only one format: `.spr`. I'll likely add more in the future.
